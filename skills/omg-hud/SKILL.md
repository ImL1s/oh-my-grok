# omg-hud

Grok has no OMC host statusline chrome. OMG HUD is a **CLI one-liner** (and JSON pack).

```bash
omg hud
omg hud --run RUN
omg hud --json
```

Example: `omg-hud: ralph|running|implement|run=abc123|‚àí`

Pair with `omg state --human` and `omg resume` for full continuity.
