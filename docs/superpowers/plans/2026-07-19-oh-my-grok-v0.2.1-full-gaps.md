**Goal:** Close every deferred residual. Where PreToolUse regex cannot hard-block, use **different mechanisms** that achieve the purpose.

## Isolation stack (not “same regex better”)

| Purpose | Primary mechanism | Secondary |
|---------|-------------------|-----------|
| Workers cannot shell external CLIs | `capability_mode: read-write` / `read-only` (no Execute) | agent `disallowedTools`; parent `--disallowed-tools` clamp |
| Leader shell still soft-guarded | PreToolUse deny (fail-open honest) | skill HARD RULES |
| Acceptance shell only via CLI | `omg accept` / frozen runner | strip OMG_ALLOW_* |
| External advisors | `omg ask` broker only (user-invoked) | child env allow only |
| Parallel without tmux | spawn_subagent worktree (default) | optional `omg ulw --fanout process` multi-PID |
